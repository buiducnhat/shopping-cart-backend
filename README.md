# SHOPPING CART backend app
## Requirements
* Node version >= 14
* npm (along with Node)
* Mongodb (version >= 4), server or localhost
## How ro run
### Change .env.local file to .env file
```mv .env.local .env```
### Edit custom PORT (for server), Secret key, and MongoDB server
### Install dependencies
```npm install```
### Run app
```node index.js```
